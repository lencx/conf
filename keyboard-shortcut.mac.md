# Keyboard shortcut

>>
⌘ : command \
⌥ : option \
⇧ : shift \
⌃ : control

## Common

| keyboard shortcut | description |
|:---:|:---|
|⌘,|Setting|
|⌘Q|Quit|
|⌘Space|Spotlight Search|
|fn⌘|Siri|
